# Project 08 — Beta Testing & User Feedback

## Purpose

Recruit beta users from the waitlist. Structure the feedback collection process. Synthesize feedback into actionable product inputs.

## Prerequisites

Complete Project 06 (product management) and Project 07 (product development). Waitlist from Project 04 is recommended.

## Global Context Files

**Loads:** `icp.md`, `product-overview.md`
Also references: `projects/04-inbound-marketing/outputs/waitlist-tracker.md`

## Skills

| Skill | Description |
|---|---|
| `beta-recruiter` | Select and invite high-ICP-tier users from the waitlist |
| `feedback-collector` | Build structured feedback survey templates |
| `feedback-synthesizer` | Tag and synthesize raw feedback into reports |
| `cs-engager` | Draft structured check-in messages for beta users |

## Outputs

- `beta-outreach-drafts/` — Beta invitation emails
- `feedback-survey-templates/` — Survey templates
- `feedback-synthesis-reports/` — Synthesized feedback reports
- `beta-user-tracker.md`
