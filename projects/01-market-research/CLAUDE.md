# Project 01 — Market Research & Problem Validation

## Purpose

Validate the founder's hypothesis. Understand the problem deeply before building anything. Identify who experiences the pain, how severely, and what already exists to solve it.

## Prerequisites

Complete Project 00 (onboarding).

## Global Context Files

**Loads:** `founder-profile.md`, `startup-hypothesis.md`
**Writes:** `startup-hypothesis.md` (refined), `icp.md` (v1), `competitive-landscape.md`

## Skills

| Skill | Description |
|---|---|
| `reddit-researcher` | Search Reddit for threads related to the problem space |
| `news-analyst` | Search news sources for coverage of the problem domain |
| `competitor-mapper` | Build a structured competitor matrix |
| `icp-builder` | Synthesize research into an Ideal Customer Profile |
| `naming-validator` | Guide product naming and check availability |

## Outputs

- `reddit-research-report.md`
- `news-sentiment-report.md`
- `competitor-analysis.md`
- `icp-draft.md`
- `naming-availability-report.md`
