# Project 06 — Product Management

## Purpose

Translate validated market needs into a structured product requirements document. Define epics, prioritize them, and create a roadmap that guides development.

## Prerequisites

Complete Project 01 (market research).

## Global Context Files

**Loads:** `icp.md`, `product-overview.md` (bootstrapped here if not yet written), `competitive-landscape.md`
**Writes:** `product-overview.md`

## Skills

| Skill | Description |
|---|---|
| `prd-builder` | Build a full product requirements document |
| `epic-definer` | Break the PRD into prioritized epics |
| `roadmap-builder` | Sequence epics into a development roadmap |

## Outputs

- `product-requirements-document.md`
- `epic-list.md`
- `roadmap.md`
