# Project 05 — Outbound Sales & Prospecting

## Purpose

Identify and reach high-fit prospects. Generate personalized outreach. Build and manage a pipeline of design partners and early customers.

## Prerequisites

Complete Project 01 (market research) and Project 02 (brand and marketing).

## Global Context Files

**Loads:** `icp.md`, `brand-voice.md`, `product-overview.md` (once available), `competitive-landscape.md`

## Skills

| Skill | Description |
|---|---|
| `prospect-researcher` | Identify prospects matching the ICP via web search |
| `email-drafter` | Produce personalized outreach email drafts |
| `pre-call-brief-builder` | Build structured pre-call briefs for meetings |
| `design-partner-advisor` | Reframe sales as design partnership conversations |
| `crm-sync-advisor` | Organize prospect data in CRM-compatible format |

## Outputs

- `prospect-list.md`
- `outreach-drafts/` — Personalized email drafts
- `pre-call-briefs/` — Pre-call prep documents
- `pipeline-tracker.md`

  `.claude/skills/` directory (available across all projects).
- **MCP builder** — Claude Code's built-in MCP server builder. Use it if you want to connect
  Claude Code to an external tool or API.

Custom skills and MCP servers you create will live in your private fork and will not affect
the upstream repo. If you build something you think would benefit other founders, see CONTRIBUTING.md
for how to suggest it.
