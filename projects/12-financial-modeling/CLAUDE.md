# Project 12 — Financial Modeling

## Purpose

Build a dynamic financial model that the founder can translate into a spreadsheet. Model revenue scenarios, cost structure, unit economics, and runway.

## Prerequisites

Complete Project 09 (pricing) and Project 06 (product management).

## Global Context Files

**Loads:** `pricing-model.md`, `product-overview.md`, `company-profile.md`

## Skills

| Skill | Description |
|---|---|
| `revenue-modeler` | Build SaaS revenue models with scenarios |
| `unit-economics-builder` | Calculate LTV, CAC, and unit economics |
| `hiring-roadmap-builder` | Plan hiring timeline and cost impact |
| `scenario-planner` | Model multiple runway and growth scenarios |

## Outputs

- `revenue-model.md`
- `cost-structure.md`
- `unit-economics.md`
- `hiring-roadmap.md`
- `runway-scenarios.md`
- `investor-metrics-summary.md`
