# Project 04 — Inbound Marketing & Content Engine

## Purpose

Build an organic content strategy and execute content across channels. Drive awareness and waitlist signups before launch.

## Prerequisites

Complete Project 02 (brand and marketing) and Project 03 (brand identity).

## Global Context Files

**Loads:** `brand-voice.md`, `brand-identity.md`, `icp.md`, `startup-hypothesis.md`

## Skills

| Skill | Description |
|---|---|
| `content-strategist` | Build a content calendar and channel strategy |
| `blog-writer` | Draft long-form blog content |
| `linkedin-writer` | Produce LinkedIn posts in multiple formats |
| `reddit-engager` | Draft authentic Reddit responses in the founder's voice |
| `youtube-scriptwriter` | Produce full video scripts with hooks and CTAs |
| `technical-report-writer` | Write technical deep-dive content |
| `waitlist-manager` | Track and qualify waitlist signups |

## Outputs

- `content-calendar.md`
- `waitlist-tracker.md`
- `blog/` — Blog post drafts
- `linkedin/` — LinkedIn post drafts
- `reddit-engagement/` — Reddit response drafts
- `youtube/` — Video scripts
- `technical-reports/` — Technical content

## Don't See What You Need?

The skills in this project cover the most common tasks for this stage, but they are not exhaustive.
If you need a capability that isn't covered, Claude Code includes two native built-in tools for this:

- **Skill creator** — Claude Code's built-in skill authoring tool. Use it to create a new custom skill
  and save it in this project's `.claude/skills/` directory (project-only) or in the root
  `.claude/skills/` directory (available across all projects).
- **MCP builder** — Claude Code's built-in MCP server builder. Use it if you want to connect
  Claude Code to an external tool or API.

Custom skills and MCP servers you create will live in your private fork and will not affect
the upstream repo. If you build something you think would benefit other founders, see CONTRIBUTING.md
for how to suggest it.
