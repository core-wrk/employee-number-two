# _dev — Private Development Notes

This directory is for the repo maintainer's use only. It is listed in `.gitignore` and will never be published to the public GitHub repo.

## Contents

- `decisions/` — Architecture Decision Records (ADRs)
- `skill-refinement-notes/` — Per-skill testing notes and iteration logs
- `project-refinement-notes/` — Per-project feedback and improvement backlog
- `roadmap.md` — Internal roadmap for repo development
- `employee-number-two-architecture.md` — Full architecture document

## Purpose

This is the right place to save:
- Notes from testing a skill with a real founder
- Observations about where the context-loader over- or under-loads
- Decisions about why a skill was scoped the way it was
- Ideas for new skills or projects that aren't ready to build yet
